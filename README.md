Detector de Perros y Gatos con TensorFlow
Este es un proyecto de aprendizaje automático de Machine Learning que utiliza TensorFlow para identificar si las imágenes proporcionadas son de perros o gatos.

Descripción
El objetivo principal de este proyecto es desarrollar un clasificador binario utilizando redes neuronales convolucionales conocidas como las famosas “CNNs” implementadas con TensorFlow. El modelo es entrenado con un conjunto de datos que contiene imágenes etiquetadas como perros o gatos dentro del repositorio, y luego este logra predecir con cierta precisión si una nueva imagen presentada contiene un perro o un gato, y de esta manera sucesivamente.
